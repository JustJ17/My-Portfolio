# Image Assets

## Logo
Please place your logo image here and name it `logo.png`

**Location:** `src/assets/images/logo.png`  
**Supported formats:** PNG, JPG, SVG  
**Recommended size:** 50px height (width will auto-adjust)  
**Fallback:** If no logo is provided, a gradient placeholder with the letter "P" will be displayed.

---

## Profile Photo
Please place your profile photo here and name it `profile-photo.jpg`

**Location:** `src/assets/images/profile-photo.jpg`  
**Supported formats:** PNG, JPG, JPEG  
**Recommended size:** 350x350px (square format)  
**Usage:** Displayed in the Hero section as a circular image  
**Fallback:** If no photo is provided, a user icon placeholder will be displayed.
